# Rencana Pengembangan Aplikasi Cloud Storage Pribadi (Alternatif Google Drive / ownCloud)

## 1. Arsitektur Sistem
- **Frontend**: ReactJS (Modern UI, dinamis, responsif)
- **Backend**: Python (FastAPI direkomendasikan karena performa tinggi untuk I/O stream)
- **Database**: MongoDB (Metadata file, info user, kuota, bandwidth limit)
- **Deployment**: Docker & Docker Compose

## 2. Fitur Utama
- **Autentikasi & Manajemen User**
  - Login, Logout (JWT Auth).
  - **Registrasi Admin-Only**: Pendaftaran user baru hanya bisa dilakukan oleh Admin melalui Control Center.
  - Pembuatan folder root otomatis di `/data/{user_id}`.
- **Manajemen File & Folder (Akses Terisolasi)**
  - CRUD (Create, Read, Update, Delete) file dan folder.
  - Setiap user hanya bisa mengakses foldernya sendiri.
- **Sistem Kuota & Bandwidth**
  - Limit penyimpanan (GB) per user (Admin-managed).
  - Limit kecepatan Download/Upload (KB/s) per user (Admin-managed).
- **Media Preview & Dashboard Admin**
  - Pratinjau foto & video tanpa download.
  - Panel kontrol admin untuk Manajemen User, Log Sistem, dan Pengaturan Limit.
